Ayuda - Gestor de Contraseñas

¿Qué es este proyecto?

Este es un sistema de gestión de contraseñas desarrollado con Django que te permite almacenar de forma segura tus credenciales de diferentes sitios web. Las contraseñas se cifran usando el algoritmo Fernet antes de guardarse en la base de datos.

Características de Seguridad

  • Cifrado: Todas las contraseñas se cifran usando Fernet antes de guardarse
  • Ocultación: Las contraseñas se muestran ocultas por defecto en el dashboard
  • Revelado bajo demanda: Solo puedes ver la contraseña cuando presionas "Mostrar"
  • Control de acceso: Solo puedes ver y editar tus propias contraseñas

Cómo usar el sistema

1. Registro e Inicio de Sesión

Crea una cuenta nueva o inicia sesión con tus credenciales existentes.

2. Agregar Contraseñas

Ve a "Agregar nueva" desde el dashboard y completa:

  • Nombre del sitio: Un nombre descriptivo (ej: "Gmail", "Facebook")
  • URL del sitio: La dirección web del sitio
  • Usuario: Tu nombre de usuario o email
  • Contraseña: Tu contraseña para ese sitio

3. Ver Contraseñas

En el dashboard verás todas tus contraseñas guardadas. Para ver una contraseña:

  • Presiona el botón "Mostrar" junto a la contraseña que quieres ver
  • La contraseña aparecerá en texto plano
  • Presiona "Ocultar" para volver a ocultarla

4. Editar Entradas

Para modificar una entrada existente:

  • Presiona el botón "Editar" junto a la entrada que quieres modificar
  • Cambia los campos que necesites
  • Si quieres cambiar la contraseña, escribe la nueva en el campo "Nueva contraseña"
  • Si no quieres cambiar la contraseña, deja ese campo vacío
  • Presiona "Guardar cambios"

Uso en Dispositivos Móviles

El sitio está optimizado para dispositivos móviles. En pantallas pequeñas:

  • Las tarjetas de contraseñas se apilan verticalmente
  • Los botones se adaptan al tamaño de pantalla
  • La navegación se mantiene accesible

Recomendaciones de Seguridad

  • Usa contraseñas únicas y fuertes para cada sitio
  • No compartas tus credenciales de acceso al gestor
  • Cierra sesión cuando termines de usar el sistema
  • Mantén actualizado tu navegador web

Soporte Técnico

Si tienes problemas con el sistema:

  • Verifica que tu conexión a internet funcione correctamente
  • Intenta recargar la página
  • Si el problema persiste, contacta al administrador del sistema
